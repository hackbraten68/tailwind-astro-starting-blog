# game idee

das game besteht aus einem runden basierten rollenspiel, wo kind und eltern sich abwechselen muessen. das kind kann manche runden nur weitermachen, wenn der erwachsene seine runde beendet oder code zum fortfahren eingibt. das game bietet soll dazu anregen, die kids gewisse sachen ueber einen laengeren zeitraum zu machen und sie so in ihren alltag einzubauen, spielerisch eben. als beispiel: es koennte eine geschichte in diesem spiel sein, den kids das all abendliche ritual des zaehneputzen einfacher zu gestalten. nehmen wir an, das kind kann an der geschichte weiterspielen, fuer eine zeit die zu spielbeginn von dem erwachsenen festgelegt wird. damit hat der erwachsene die kontrolle darueber wie das spiel und der vermeintliche gang ins bad spielerisch gestaltet werden kann. geht man von 15min aus, kann das kind bestimmte teile der geschichte spielen, sei es farmen, idlen, platformer oder was auch immer - was eben passend fuer das thema ist. nach ablauf des timers als trigger, kommt die meldung das es nun erst weitergeht wenn das kind die zaehne geputzt hat, somit erschaffen wir eine neue routine und trainieren das muskelgedaechtnis. hat das kind alles erledigt, kann der erwachsene (nennen wir diesen fall von spiel, den "i need to get shit done" modus) den code eingeben, damit die geschichte weiter gespielt werden kann. 

## game modi

- "i need to get shit done" mode, parent-passive, der erwachsene muss nur codes eingeben um dem kind neue teile der geschichte freizuschalten.
- active-mode, wo erwachsener und kind im 2-player-mode gemeinsam questen gehen.

## game adventures

aehnlich wie quests sollen adventures aber einzelne fuer sich abgeschlossene quest-reihen sein, die eine gewisse max. dauer haben koennen. als beispiel zaehneputz adventure wo das kind eine fanatasie welt abtauchen soll die sich mit dem thema zaehne putzen auseinandersetzt und spielerisch erklaert ueber die quest reihe wieso es wichtig ist. dann farming ueber hack n slay um belohnungen frei zu schalten. wichtig ist das der erwachsene entweder mitspielen kann oder nur per codes passiv am rundenbasierten questing teilnimmt. 

## game sections (main und neben quests)

das spiel soll taeglich nur 30-60min gespielt werden. quest-reihe fuer den tag soll aus main und neben-quests bestehen.

### Spielmechanik:

1. **Charakterentwicklung:**
   - Das Kind kann einen eigenen Charakter erstellen und durch das Erfüllen von Aufgaben und Herausforderungen im Spiel Fortschritte machen.
   - Jeder Charakter hat Fähigkeiten und Belohnungen, die sich auf reale Aufgaben im Alltag beziehen, z. B. Zähneputzen, Aufräumen oder Hausaufgaben.

2. **Abwechslungsreiche Aufgaben:**
   - Die Geschichte im Spiel kann verschiedene Abschnitte haben, die jeweils mit verschiedenen Aktivitäten verknüpft sind. Zum Beispiel könnte das Kind auf einer virtuellen Farm arbeiten, um "Ressourcen" zu sammeln.
   - Diese Ressourcen können dann verwendet werden, um im Spiel voranzukommen oder den Charakter zu verbessern.

### Elternmodus:

1. **Zeitmanagement:**
   - Der Elternmodus ermöglicht es, die Spielzeit des Kindes zu begrenzen und festzulegen, wann bestimmte Aktivitäten im Spiel durchgeführt werden sollen.
   - Der Timer kann für jede Aufgabe individuell eingestellt werden, um den Fortschritt des Kindes zu steuern.

2. **Belohnungssystem:**
   - Eltern können Belohnungen im Spiel festlegen, die das Kind erhält, wenn es reale Aufgaben erledigt hat. Dies kann zusätzliche Spielzeit, neue Fähigkeiten oder kosmetische Gegenstände für den Charakter im Spiel umfassen.

### Integration von Alltagsroutinen:

1. **Zauberhafte Geschichten:**
   - Die Geschichten im Spiel können magische Elemente enthalten, die die reale Welt mit der virtuellen verbinden. Zum Beispiel könnte das Zähneputzen im Spiel dazu führen, dass der Charakter im Spiel magische Zähne erhält.

2. **Echtzeit-Interaktion:**
   - Das Spiel kann in Echtzeit aktualisiert werden, um Eltern zu informieren, wenn das Kind eine Aufgabe im Spiel abgeschlossen hat und bereit ist, mit der nächsten Aktivität im Alltag fortzufahren.

### Fortschritt und Langzeitmotivation:

1. **Langfristige Ziele:**
   - Das Spiel könnte langfristige Ziele für das Kind enthalten, die nur durch regelmäßige Teilnahme und das Erfüllen von realen Aufgaben erreicht werden können.

2. **Community-Aspekt:**
