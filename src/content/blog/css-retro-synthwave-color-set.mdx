## Let's make some gradients, because we all love them right ?! üåäüçáüçßüå∫

okay, now let's see how these colors could nicely work together as a linear-gradient. i mostly use [cssgradient.io](https://cssgradient.io/), so you guys can freak out by yourself with these colors.

### Three-colored

#### Deeppink -> Blue Violet -> Black
![deeppink to blue violet to black](/images/blog/pink-blueviolet-black.png)

```javascript
background: rgb(255,20,147);
background: linear-gradient(90deg, rgba(255,20,147,1) 0%, rgba(138,43,226,1) 50%, rgba(0,0,0,1) 100%);
```

#### Deeppink -> Aqua -> Navy Blue
![deeppink to aqua to blue navy](/images/blog/deeppink-aqua-navyblue.png)

```javascript
background: rgb(255,20,147);
background: linear-gradient(90deg, rgba(255,20,147,1) 0%, rgba(56,239,245,1) 66%, rgba(0,0,139,1) 100%);
```

#### Blue Violet -> Deeppink -> Aqua
![deeppink to blue violet to black](/images/blog/blueviolet-deeppink-aqua.png)

```javascript
background: rgb(138,43,226);
background: linear-gradient(90deg, rgba(138,43,226,1) 0%, rgba(255,20,147,1) 33%, rgba(56,239,245,1) 100%);
```

#### Purple -> Blue Violet -> Deeppink 
![purple to blue violet to deeppink](/images/blog/purple-blueviolet-deeppink.png)

```javascript
background: rgb(138,43,226);
background: linear-gradient(90deg, rgba(138,43,226,1) 0%, rgba(255,20,147,1) 33%, rgba(56,239,245,1) 100%);
```
